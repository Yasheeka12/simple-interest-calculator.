# Simple Interest Calculator

## Description
This project explains how to calculate Simple Interest using a simple formula.
It is easy to understand and suitable for beginners.

## Formula
Simple Interest (SI) is calculated as:

SI = (Principal × Rate × Time) / 100

## Variables
- Principal (P): Initial amount of money
- Rate (R): Rate of interest per year
- Time (T): Time period in years

## Example
If:
- Principal = 1000
- Rate = 5
- Time = 2

Then:
SI = (1000 × 5 × 2) / 100 = 100

This README file contains the details of a Simple Interest Calculator.
